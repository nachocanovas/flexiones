# Flexiones 365 🔥

Reto de flexiones: 1 el día 1, 2 el día 2... hasta 365.

## Subir a Vercel (3 pasos)

### Opción A — Sin código (más fácil)

1. Ve a [vercel.com](https://vercel.com) y crea una cuenta gratis
2. Haz clic en **"Add New Project"**
3. Elige **"Upload"** y arrastra esta carpeta entera
4. Vercel detecta Vite automáticamente → pulsa **Deploy**
5. En ~1 minuto tienes tu URL (ej: `flexiones-365.vercel.app`)

### Opción B — Con GitHub (recomendado para updates)

1. Sube esta carpeta a un repositorio en [github.com](https://github.com)
2. En Vercel → "Add New Project" → conecta tu GitHub
3. Selecciona el repo → Deploy
4. Cada vez que actualices el código en GitHub, Vercel redespliega solo

## Añadir al iPhone como app

1. Abre la URL de Vercel en **Safari** (importante: Safari, no Chrome)
2. Pulsa el botón compartir (□↑) en la barra inferior
3. Selecciona **"Añadir a pantalla de inicio"**
4. Nombre: "Flexiones 365" → **Añadir**

Ya aparece como app en tu pantalla de inicio, a pantalla completa, sin barra de Safari.
